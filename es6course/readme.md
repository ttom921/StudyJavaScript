### espassport-test 這是測試有關`html-bundler`的用法
作者將常用的前端的打包工具，寫成工具來使用[github](https://github.com/be-fe/html-bundler)
全域安裝

```
npm install html-bundler -g
```
建立專案

```
hb create espassport -w 
```

執行專案

```
npm install && hb dev
```
